# TournaMagic App Idea

## What this app is about
TournaMagic is a web app for friends who play **Magic: The Gathering draft tournaments**.

The app helps a small group (for example 4–12 players) run a full tournament without paper notes.

## Main idea
1. Add all players at the start.
2. The app creates a **round-robin** schedule (everyone plays everyone).
3. Each match is **best of three** (Bo3), so each match has one winner.
4. Enter match results quickly.
5. See a live leaderboard that updates automatically.

## Why it exists
When friends play at home, tournament tracking can be confusing:
- Who still has to play?
- Who is currently first?
- What is each player’s match score?

TournaMagic answers these questions in one place with a clear UI.

## Current product goals
- Easy tournament setup
- Fast result entry during play
- Live ranking and progress tracking
- Local Docker startup for simple use
- Persistent backend data so tournaments are not lost
