# TournaMagic

Phase 1-3 implementation of the MTG draft tournament tracker using Vue 3 + Vite + TypeScript.

## Implemented phases
- **Phase 1:** Frontend foundation, routing, Pinia store setup, and MTG-inspired theme tokens.
- **Phase 2:** Domain logic for round-robin pairings, Bo3 validation, and leaderboard ranking with tie-breaks.
- **Phase 3:** Setup flow, tournament dashboard, result entry/edit, filtering, progress view, and LocalStorage persistence.

## Run locally
```bash
npm install
npm run dev
```

## Run tests
```bash
npm test
```
